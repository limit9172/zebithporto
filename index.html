Proyek

Kalkulator Python

Proyek ini merupakan sebuah kalkulator interaktif berbasis CLI yang dibuat menggunakan bahasa pemrograman Python. Dengan antarmuka sederhana, pengguna dapat melakukan operasi matematika dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian.

IP Tracer

IP Tracer adalah script Python yang mampu melacak lokasi geografis dari sebuah alamat IP menggunakan layanan API eksternal. Informasi yang diperoleh meliputi negara, kota, ISP, koordinat latitude-longitude, dan lainnya.

Phishing

Tools phishing ini dirancang untuk edukasi dan simulasi keamanan, dengan tujuan menunjukkan bagaimana serangan rekayasa sosial dapat terjadi.

Xenostresser

Xenostresser adalah tools serangan DDoS yang dikembangkan untuk menguji ketahanan server terhadap serangan lalu lintas palsu secara masif.

Zentich AI

Zentich AI adalah proyek berbasis Python dan machine learning yang dirancang untuk melakukan prediksi dan klasifikasi data.

Xein Tools

Xein Tools adalah kumpulan alat hacking dan OSINT dalam satu paket terminal.

Tic-Tac-Toe Game

Tic-Tac-Toe klasik ini dikembangkan dengan HTML, CSS, dan JavaScript. Game ini memungkinkan dua pemain bermain secara bergiliran di satu perangkat.
